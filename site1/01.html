כח הצבים

מבוא:

בשיעור הזה אנחנו הולכים להכיר את שפת התכנות פייתון. זאת שפה שמאוד נוח לקרוא אותה ולכן היא טובה מאוד למתכנתים מתחילים, ומצד שני היא שפה חזקה שאפשר לעשות איתה את כל הדברים המתקדמים יחסית בקלות. משתמשים בפייתון בגוגל, ב-NASA, ב-dropbox ובחברות וארגונים נוספים רבים מאוד.

שלב 0: מפעילים את עורך הטקסט שלנו ב-IDLE

אם פייתון עדיין לא מותקן על המחשב שלכם, כעת זה הזמן להתקין אותו (בגרסה 3) מהאתר הרשמי

עכשיו שפייתון מותקן, הגיע הזמן לפתוח את סביבת הפיתוח שמגיעה עם פייתון - IDLE

  • ב- Windows, תוכלו למצוא את IDLE בתפריט התחל.
  • במקינטוש או לינוקס, תפתחו חלון Terminal, תקלידו idle3 ותלחצו enter.

כש-IDLE ייפתח, אתם תראו את החלון הראשי שלו שנקרא Python Shell. בחלון הזה אנחנו נוכל לראות תוצאות של מה שאנחנו עושים ואולי גם כל מיני שגיאות, אבל החלון העיקרי שנבלה בו זמן הוא של עורך טקסט. כדי לפתוח חלון טקסט לחצו על File -> New Window ואז תסדרו כך שתוכלו לראות את שני החלונות.

עכשיו אתם מוכנים להתחיל.

שלב 1: שלום, עולם!

צעדים

  1. פתחו את עורך הטקסט שלכם, וכתבו בו את שורת הקוד הבאה:

    print("Hello world!")

  2. שמרו את הקובץ בשם hello.py (מקובל תמיד לתת לקבצי פייתון סיומת של py).

  3. עכשיו, בואו נגרום לפייתון (לפייתון?...) להריץ את הקוד שלכם. בחרו בתפריט Run -> Run Module

שימו לב, אתם יכולים לשנות את הטקסט בתוך המרכאות לכל מה שבא לכם. תנסו למשל לגרום לפייתון להדפיס את השם שלכם.

Step 2: שלום, צב!

עכשיו אנחנו הולכים לשחק קצת עם צבים. צב הוא רובוט קטנטן שיכול לצייר ציורים על גבי המסך שלכם. אנחנו נשתמש בקוד פייתון כדי לגרום לו לזוז ולצייר.

  1. פתחו חלון קוד חדש (מתוך ה- File Menu) וכתבו את שורות הקוד הבאות:

    from turtle import *
shape("turtle")

forward(100)

done()

  2. שמרו את התוכנה שלכם (כן, תוכנה! כי אתם עכשיו מתכנתים) בשם myturtle.py ובחרו Run -> Run Module. רואים איך הצב זז קדימה 100 פיקסלים (נקודות על המסך)? לצב מחובר עט שמצייר קוים כשהצב זז על גבי המסך.

    תזכורת: שמות של קבצי פייתון תמיד צריכים להסתיים ב- "py".

  3. בואו נגרום לצב לזוז על פני המסך! נסו להשתמש ב- forward(200) וגם לסובב את הצב ע"י שתשמשו ב- right(90) וב- left(90). לדוגמא, backward(20) אומר לצב לזוז אחורה 20 פיקסלים, ו- right(90) אומר לצב לפנות 90 מעלות ימינה. אתם יכולים לתת לצב כמה הוראות שתרצו, והצב פשוט יבצע אותן לפי הסדר שבו כתבתם אותן. 

    from turtle import *
shape("turtle")

forward(100)
right(120)
forward(100)
left(90)
backward(100)
left(90)
forward(50)

done()

זויות ומעלות

שחקו קצת עם הקוד ותגרמו לצב לעשות כל מיני צורות בעזרת שימוש ב- forward, backward, left, right.

קבלו טיפ: זכרו, forward ו- backward מזיזים את הצב כמות פיקסלים מסויימת, אבל left ו- right מסובבים אבל לא מזיזים אותו.

בואו נסתכל על צב שפונה ימינה. 


          צפון
           0
           |
  מזרח     |     מערב
  270 -----+----- 90
           |
           |
          180
         דרום

נהוג למדוד סיבובים של צבים בעזרת מעלות של זוית. כמו שאתם אולי מכירים מלימודי גיאומטריה, סיבוב שלם הוא 360 מעלות ופנייה של 90 מעלות היא רבע סיבוב. כלומר, נניח שהצב מסתכל צפונה, אם נסובב אותו ב- 90 מעלות ימינה, הוא יסתכל מזרחה, אם נסובב אותו ב- 180 מעלות הוא יסתכל דרומה (בדיוק לכיוון ההפוך). אם נסובב אותו 270 מעלות הוא יסתכל מערבה. ואם נסובב אותו ב- 360 מעלות, נגרום לו לעשות סיבוב שלם ולחזור להיות בדיוק כמו שהוא היה.

ומה בנוגע לפניה שמאלה?


          צפון
           0
           |
  מזרח     |     מערב
  90 -----+----- 270
           |
           |
          180
         דרום

פנייה שמאלה היא בדיוק ההיפך - כשהצב מסתכל צפונה, אם נסובב אותו ב- 90 מעלות שמאלה, הוא יסתכל מערבה. אם נסובב אותו ב- 180 מעלות הוא יסתכל דרומה. אם נסובב אותו 270 מעלות הוא יסתכל מזרחה. ואם נסובב אותו ב- 360 מעלות, נגרום לו להישאר כמו שהוא היה. פניה של 360 מעלות היא סיבוב מלא.

מה עושה הקוד שבתחילת ובסוף התכנה שלנו

from turtle import *, נותן לנו להשתמש בכל הכלים של ספריה בשם turtle שהיא חבילה של קוד שבה אנחנו יכולים להשתמש כדי לצייר על המסך. בתכנות, ספריה זה קובץ קוד שמישהו אחר כתב ואנחנו יכולים להשתמש בו כתשתית בשביל הקובץ שלנו במקום לכתוב את הכל מאפס. שימוש בספריות מאפשר לנו לחסוך זמן ולהתמקד יותר בדברים שמעניינים אותנו.

shape זאת פעולה שמשנה את הצורה של הדמות שאנחנו מזיזים. אנחנו נעדיף להשתמש בצורה "turtle", אבל אפשר גם לבחור: "arrow", "circle", "square", "triangle" או "classic".

done זאת פקודה שאומרת לפייתון שסיימנו לעבוד שאר הפקודות של turtle. בעיקרון היא לא עושה דום דבר מעניין, אבל אם נשכח אותה, זה יכול לבלבל את החלונות של IDLE.

שימו לב שאנחנו נוסיף את השורות האלו לכל התוכנות שלנו, אבל לא נתמקד בהן יותר

צעד 3: מציירים צורות!

בואו נגרום לצב שלנו לצייר ריבוע.

  1. פתחו קובץ חדש בעורך הטקסט שלכם, וכתבו בו את שורות הקוד הבאות:

    from turtle import *
shape("turtle")

forward(100)
right(90)
forward(100)
right(90)
forward(100)
right(90)
forward(100)
right(90)

done()

    שמרו את התכנה שלכם ובחרו ב- run -> new module. רואים ריבוע? הצב פונה 90 מעלות ארבע פעמים, ומסיים כשהוא מסתכל בדיוק לאותו הכיוון שממנו התחיל. זה בגלל שכשפונים 90, 90, 90 ו- 90 שוב, הצב מסתובב 360 מעלות בסך הכל.

    מה עם משולש? למשולש יש שלוש פינות. זה אומר שנצטרך לסובב את הצב שלנו 3 פעמים. אם אנחנו רוצים לסיים באותו הכיוון שממנו התחלנו, אנחנו נצטרך לסובב את הצב סך הכל 360 פעמים, כמו שראינו במקרה של הריבוע. 360 חלקי 3 שווה 120. לכן אנחנו נסובב את הצב 120 מעלות, ואז 120 מעלות, ואז 120 מעלות פעם שלישית.

  2. תנסו לערוך את הקוד שלכם כדי שהצב יצייר משולש. תריצו את הקוד שלכם. קיבלתם משולש? אם לא, תנסו אולי להתייעץ עם חברים


אתגר - האם תוכלו לצייר את הבית הזה? האם תוכלו לעשות את זה מבלי לחזור על אותו קו פעמיים?

בחרו צבע

מהו הצבע האהוב עליכם? אתם יכולים לשנות את הצבע של הקווים שהצב מצייר על ידי שימוש בפונקציה pencolor, ואתם יכולים גם לשנות את עובי העט ע"י pensize:

  1. ערכו את הקוד שלכם כדי שיראה כמו הדוגמא הבאה:

    from turtle import *
shape("turtle")

pensize(10)
pencolor("red")
forward(100)
right(120)

pencolor("blue")
forward(100)
right(120)

pencolor("green")
forward(100)
right(120)

done()

  2. תריצו את הקוד. מה הצב צייר על המסך?
    הקוד הזה מצייר משולש עבה בשלושה צבעים שונים.

  3. יש עוד המון צבעים אחרים שאפשר להשתמש בהם, נסו לחשוב על שמות של צבעים, לשנות בקוד, להריץ ולראות אם זה עובד.

שלב 4: בואו נחזור על זה (בעזרת לולאות)

בתוכנית הקודמת שעשינו, כתבנו את אותן השורות בדיוק הרבה פעמים. כמובן שיש דרך יותר חכמה לעשות את זה וקוראים לזה לולאה. כשרוצים לחזור על אותן שורות קוד מספר פעמים ידוע מראש, משתמשים בפייתון בלולאת for.
במקרה של הריבוע שלנו, אנחנו רוצים לחזור על אותן פעולות 4 פעמים (פעם לכל צלע).

דוגמה ראשונה ללולאה (עדיין לא ננסה להבין עד הסוף איך לולאות עובדות)

  1. צרו קובץ חדש וכתבו בו את שורות הקוד הבאות:

    from turtle import *
shape("turtle")

for count in range(4):
    forward(100)
    right(90)

done()

  2. שמרו את הקובץ ותריצו אותו.

    שימו לב ששורות הקוד שבתוך הלולאה מוזזות שמאלה ביחס לשורות שמעל. ההזזה הזאת, שנקראת אינדנטציה (indentation או TAB או בעברית הזחה), היא הדרך לקבוע בפייתון איזה חלק מהקוד הוא בתוך הלולאה. מקובל בפייתון לעשות אינדנטציה באמצעות 4 רווחים ואפשר להשתמש בכפתור TAB בשביל ליצור אותה וב Shift+TAB בשביל לבטל את האינדנטציה.

  3. בואו נראה מה יקרה עם נזיז שמאלה רק את השורה forward, תשנו את התוכנה שלכם, כך שיופיע הקוד הבא: 

    from turtle import *
shape("turtle")

for count in range(4):
    forward(100)
right(90)

done()

  4. תנסו לנחש מה יקרה, ואז תריצו את הקוד ותבדקו אם צדקתם.

  5. אוקי, עכשיו כשראינו מה התפקיד של האינדנטציה, בואו נתקן את האינדנטציה ונשנה את הקוד חזרה למה שהוא היה קודם. עכשיו בואו תנסו לשנות את הקוד כך שייצור צורות אחרות - משולש, מחומש, משושה, מתומן. תנסו לשחק עם שלושת המספרים - מה שבתוך range, right ו- forward.

    אתגר לסיום - האם תצליחו לגרום לצב לצייר משהו שנראה כמו עיגול?

הערה: אם אתם מעדיפים לרוץ קצת קדימה בעצמכם, כאן תוכלו למצוא את הדוקומנטציה המלאה של ספריית turtle
