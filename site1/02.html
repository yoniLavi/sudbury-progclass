שלב 5: שמים דברים במשתנים

ברוכים הבאים לשיעור השני! הפעם נמשיך לעבוד עם הצב, אבל נלמד להשתמש במושגים ועקרונות נוספים של תכנות בשביל לעשות סדר ולעזור לנו לעשות דברים יותר מתקדמים.

בואו נמשיך לעבוד על הריבוע מהתרגיל הקודם, אבל הפעם, במקום לרשום את המספרים עמוק בקוד, נשתמש במילים. אנחנו נגדיר את המילה sides שתחזיק את מספר הצלעות, את המילה length שתחזיק את אורך הצלע ואת המילה angle שתחזיק את גודל הזוית.
מילים כאלו שאנחנו שמים בהן ערכים נקראות משתנים (variables) והן עוזרות לנו לסדר את התוכנה כך שיהיה יותר נוח להבין מה קורה ויותר קל לשנות את ההתנהגות. משתנים הם מאוד שימושיים בתכנות ואנחנו נדבר ונשתמש בהם עוד המון בהמשך.

  1. תשנו את הקובץ, כך שיהיה כתוב:

    from turtle import *
shape("turtle")

sides = 4
length = 100
angle = 90
for count in range(sides):
    forward(length)
    right(angle)

done()

  2. שמרו את התכנה שלכם ותריצו ע"י- run -> new module.

תרגיל: ציירו צורות אחרות

תוכלו לצייר את המצולעים האלו ע"י כך שתשנו את המספרים במשתנים?

  • משולש? (3 צלעות)
  • מחומש? (5 צלעות)
  • משושה? (6 צלעות)
  • מתומן? (8 צלעות)

שימו לב, למשולש יש 3 צלעות (צדדים) ובסיבוב בין הצלעות נצטרך לפנות 120 מעלות (120 מעלות 3 פעמים זה בדיוק סיבוב שלם - 360 מעלות). לריבוע יש 4 צלעות, ופונים 90 מעלות כל פעם (גם הפעם, כמו תמיד, סך כל הסיבוב הוא 360 מעלות). למשושה יש 6 צלעות, כמה מעלות צריך לפנות בשביל לצייר אותו? תנסו לחשב או אולי פשוט לנסות לשים מספרים שונים.

שלב 6: מסתובבים

במקום לחשב את הזויות בעצמנו, למה שלא ניתן למחשב לעשות את העבודה בשבילנו? פייתון יכול לעשות כל מיני פעולות על מספרים כגון חיבור, חיסור, כפל וחילוק.

למשל, במקום לכתוב sides = 4 אנחנו יכולים להשתמש בחיבור ולרשום sides = 3 + 1 או להשתמש בחיסור sides = 5 - 1. כפל וחילוק הם דומים: בשביל כפל, משתמשים בפייתון בסימן * (כוכבית) ובשביל חילוק משתמשים בסימן / (סלאש).

בכל צורה, אנחנו צריכים בסך-הכל לפנות 260 מעלות, אז בריבוע צריך לפנות 360 / 4 מעלות ובמשולש 360 / 3 מעלות.

בואו ננסה את זה

  1. תשנו את התוכנה כך שתחשב את הזוית בעצמה:

    from turtle import *
shape("turtle")

sides = 4
length = 20
angle = 360 / sides

for count in range(sides):
    forward(length)
    right(angle)

done()

  2. תריצו ותראו שקיבלתם ריבוע. עכשיו תנסו לשנות את מספר הצלעות (sides) כל שתקבלו צורה אחרת. תנסו למשל לצייר כך מתומן (8 צלעות).

שלב 7: צובעים את הצורות

  1. אנחנו יכולים לבקש מהצב שימלא את הצורה שאנחנו מציירים בצבע בעזרת הפעולות begin_fill ו- end_fill. תוסיפו את הפקודות האלו לקוד שלכם כך: 

    from turtle import *
shape("turtle")

sides = 4
length = 20

fillcolor('red')
pencolor('red')
begin_fill()

angle = 360/sides
for count in range(sides):
    forward(length)
    right(angle)

end_fill()

done()

    כמו עם pencolor מהשלבים הקודמים, fillcolor קובע את הצבע שבו ימולאו הצורות שלכם באמצעות שמות הצבעים.

    שימו לב שאתם צריכים להשתמש ב-begin_fill כדי להגיד לצב שאתם מתחילים לצבוע וב- end_fill כדי להגיד לו שסיימתם לצייר את הצורה.

  2. תנסו לשנות את הצבעים, הצורות והאורכים ולצייר כך כל מיני דברים אחרים.

שלב 8: מרימים את העט

אם בא לכם להזיז את הצב למקום חדש מבלי להשאיר סימן, אתם יכולים להרים את העט של הצב בעזרת penup ואז לחזור שוב לצייר עם pendown.

  1. הנה דוגמא:

    from turtle import *
shape("turtle")
pencolor('red')

for count in range(20):
    penup()
    forward(10)
    pendown()
    forward(20)

done()

  2. אתם אמורים לקבל קו מקווקו.

חזרה הביתה

טריקים נוספים: home מחזיר את הצב לנקודת ההתחלה. clear מנקה את כל המסך, ו-reset עושה את שניהם (מאפס את הכל).

שלב 9: מגדירים פקודות חדשות

בשלב הזה אנחנו ניקח את התוכנה שלנו שמציירת ריבוע ונמשיך לעבוד עליה כך שתהיה אפילו יותר מסודרת ומוצלחת!

  1. תזינו שוב את הקוד הבא שכבר כתבנו בשלב 5 ובו נעזרנו במשתנים בשביל לעשות סדר:

    from turtle import *
shape("turtle")

sides = 4
length = 100
angle = 360/sides

for n in range(sides):
    forward(length)
    right(angle)

done()

  2. תריצו ותוודאו שקיבלתם את אותו הריבוע כמו קודם.

  3. זאת תוכנה קצת ארוכה, אבל היא מוצלחת כי אנחנו ראינו שבזכות המשתנים אפשר לצייר איתה בקלות גם צורות אחרות. הבעיה היא שאם אנחנו רוצים לצייר כמה צורות, אנחנו צריכים להעתיק אותה שוב ושוב לכל צורה. למזלנו, יש דרך לחסוך לנו את העבודה הזאת - נגדיר פונקציה.
    כמו שבשלב 5 נעזרנו במשתנים בשביל לתת שמות למספרים שלנו, הפעם אנחנו ניתן שם לפעולה. באופן כללי, בתכנות, נרצה לעיתים קרובות לארוז מספר פעולות תחת שם אחד בשביל לחסוך מקום ולעשות סדר וכדי לעשות זאת, אנחנו צריכים ליצור פונקציה (function - פעולה).

  4. תערכו את הקוד ותוסיפו את השורה def poly(): . מתחת לשורה הזאת, תכניסו את שורות הקוד שהיו לנו קודם כדי לצייר את הריבוע עם אינדנטציה (אפשר עם לחיצה על TAB) - בדיוק כמו בלולאה: 

    from turtle import *
shape("turtle")

def poly():
    sides = 4
    length = 100
    angle = 360/sides

    for n in range(sides):
            forward(length)
            right(angle)

pencolor('red')
poly()
right(180)
poly()

done()

  5. תריצו את התוכנה ותקבלו שני ריבועים.

מה שבעצם קרה כאן זה שחסכנו לעצמנו הרבה זמן, על-ידי כך שהגדרנו פעולה (פונקציה) חדשה בשם poly. אחרי הגדרת הפונקציה, כל פעם שכתבנו poly(), מה שפייתון עשה זה להריץ בדיוק את מה שכתבנו בתוך האינדנטציה מתחת לשורת ה-def poly():.

פונקציות הן דרך נהדרת לחסוך לנו כתיבה ולעשות את הקוד מסודר. תנסו לשחק בעצמכם עם הדוגמא ולראות מה קורה כשמשנים את הפונקציה.
